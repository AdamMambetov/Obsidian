# Автозагрузка

## Windows

```
C:\Users\<USER_NAME>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
```

Путь по которому расположены приложения, которые запускаются вместе с [[Windows]].

Альтернативный способ:

```
Win + R
```

```
shell:startup
```
