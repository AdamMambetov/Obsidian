# Субфакториал

Субфакториал - это количество беспорядков данного числа, т.е. сколько раз можно переставить вещи не ставя их на одни и те же места и не ставя число на свой номер.


|  n  |  !n |
|:---:| ---:|
|  0  |   1 |
|  1  |   0 |
|  2  |   1 |
|  3  |   2 |
|  4  |   9 |
|  5  |  44 |
