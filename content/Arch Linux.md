## Pacman

> [!note]- Обновить все пакеты
> ```bash
> sudo pacman -Syu
> ```

> [!note]- Скачать пакет
> ```bash
> sudo pacman -Sy package_name
> ```

> [!note]- Поиск пакетов по названию
> ```bash
> sudo pacman -Ss package_name
> ```

> [!note]- Удалить пакет
> ```bash
> sudo pacman -R package_name
> ```
